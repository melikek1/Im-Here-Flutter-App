# I'M HERE: Accessibility App for Hearing Impaired

This project is a mobile application developed using the Flutter framework. The main goal of "I'M HERE" is to provide a simple, high-contrast digital interface for hearing-impaired individuals to communicate their basic and urgent needs in public places like hospitals or markets.

## Project Motivation
I developed this app to solve a real-world communication barrier. In crowded or stressful environments, quickly expressing a specific need (like "I need a doctor") can be difficult. This app uses large-scale typography and clear visuals to bridge that gap.

## Development Details
* **Framework:** Flutter (stable version)
* **Language:** Dart
* **Design Process:** The UI flow was initially sketched and prototyped in Figma to ensure accessibility standards were met before coding.
  
## UI/UX Design & Prototyping
Before the coding phase, I designed a high-fidelity prototype to test the user flow and accessibility features. You can access the interactive design and see how the app's interface was planned through the link below:

## Application Structure
- **Main Interface:** A simple grid-based navigation system.
- **Emergency Action:** A dedicated button for immediate assistance.
- **Category Modules:** Custom views for different scenarios (Medical, Transportation, Shopping).

## Installation for Review
If you have the Flutter SDK installed, you can run the project by following these steps:
1. Download or clone the repository.
2. Run `flutter pub get` in the terminal to install the necessary packages.
3. Use `flutter run` to launch the app on an emulator or physical device.
